# Contributing Guide

This repository is maintained for DAY NIGHT DELIVERY SERVICES.

## Working Rules

1. Create clear commits.
2. Do not upload passwords, API keys, or private files.
3. Keep all business content professional and accurate.
4. Keep Arabic and English content encoded as UTF-8.
5. Test files locally before pushing.
6. Use branches for major changes.

## Commit Message Examples

```text
Add WordPress pricing page
Update UAE delivery coverage
Fix Arabic encoding
Add Supabase order tracking plan
Improve README documentation
````

## Before Push

Run:

```powershell
git status
git add .
git commit -m "Your clear message"
git push
```
