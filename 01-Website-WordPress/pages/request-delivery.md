# Request Delivery

## اطلب توصيل الآن

يمكنك طلب خدمة توصيل من **DAY NIGHT DELIVERY SERVICES** بسهولة عبر الهاتف أو الواتساب.

---

## البيانات المطلوبة

يرجى إرسال البيانات التالية:

### بيانات المرسل

- اسم المرسل.
- رقم هاتف المرسل.
- عنوان الاستلام.
- موقع الاستلام على الخريطة.

### بيانات المستلم

- اسم المستلم.
- رقم هاتف المستلم.
- عنوان التسليم.
- موقع التسليم على الخريطة.

### بيانات الشحنة

- نوع الشحنة.
- عدد القطع.
- وزن الشحنة إن وجد.
- هل يوجد دفع عند الاستلام؟
- ملاحظات إضافية.

---

## خطوات الطلب

1. أرسل البيانات عبر الواتساب.
2. يتم مراجعة الطلب وتأكيد السعر.
3. يتم تحديد وقت الاستلام.
4. يتم استلام الشحنة.
5. يتم توصيل الشحنة ومتابعة حالتها حتى التسليم.
